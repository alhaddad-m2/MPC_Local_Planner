# MPC_Local_Planner
Local Planner for a differential drive mobile robot using MPC and Acados with ROS Implementation. It has been tested on Husky Mobile Robot.

## Prerequisites
- [Acados](https://github.com/acados/acados)

## Tested
- Ubuntu 20.04 , 18.04
